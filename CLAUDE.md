# Summit Project Instructions

## Product
- App name: Summit
- Tagline: Earn your edge
- Brand type: elite finance training ground
- Users: ambitious students and early-career professionals
- Goal: build a premium, sticky finance training app centered on daily judgment challenges

## Brand
- Dark, premium, minimal, modern
- No childish visuals
- No cartoon mascots
- No "financial literacy" tone
- Voice: confident, crisp, intelligent, disciplined, motivating

## Product language
- Lessons = Challenges
- Levels = Ranks
- XP = Edge Points
- Streak = Run
- Categories = Tracks
- Weekly events = Expeditions

## Rank ladder
- Rise
- Ascent
- Ridge
- Peak
- Summit

## MVP screens
1. Welcome
2. Track Selection
3. Home
4. Challenge
5. Results
6. Tracks
7. Leaderboard

## Core loop
Home → Challenge → Results → Home

## UI priorities
- Mobile-first
- Fast, clean, premium
- Strong hierarchy
- Minimal clutter
- Reusable components
- Build Home, Challenge, and Results first if prioritization is needed

## Home screen structure
- Top bar with Summit wordmark, rank badge, profile icon
- Hero card for Today's Challenge
- Progress strip with Run, Edge Points, Rank, Top %
- Rank progress card called "Your Climb"
- Tracks section
- Expedition preview
- Leaderboard preview
- Bottom nav: Home, Challenges, Tracks, Leaderboard, Profile

## Challenge screen structure
- Header with track and back button
- Challenge title and metadata
- Scenario card
- Multiple choice answers
- Primary CTA: Lock In Answer

## Results screen structure
- Result banner: Correct or Missed It
- Right answer card
- Explanation card
- Reward section with Edge Points and Run
- Performance insight
- CTAs: Back to Home and Continue Track
